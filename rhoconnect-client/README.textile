h1. RhoConnect-Client

The "RhoConnect-Client":http://github.com/rhomobile/rhodes/rhoconnect-client is a framework for building client application for RhoConnect.

RhoConnect-Client is available as an open source product under the MIT license.  For developers interested a commercial license is still "available":http://rhomobile.com/products/rhodes/license/.
