Introduction

Thank you for installing Rhodes and RhoSync!

Rhodes is a framework for building locally executing, device-optimized mobile applications. These applications are optimized for interacting with transactional enterprise application backends. It is also designed to work with synced local data using a local database such as SQLite or HSQLDB and a generic backend synchronization framework such as RhoSync, another open source component available from Rhomobile. Rhodes is available for iPhone, Research in Motion (Blackberry), Windows Mobile, and Android phones. For more information about Rhodes and RhoSync, please see the Rhomobile Docs.
